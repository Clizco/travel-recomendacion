# TravelBloom - Sitio web de recomendaciones de viajes

Este proyecto es un sitio web interactivo creado con HTML, CSS y JavaScript. Permite buscar recomendaciones de destinos turísticos según preferencias como playas, templos o países.

## Páginas incluidas

- Página de inicio
- Página Quiénes somos
- Página de contacto

## Funcionalidades

- Barra de navegación funcional
- Buscador interactivo con JavaScript
- Recomendación de playas con dos imágenes
- Recomendación de templos con dos imágenes
- Recomendación basada en países con dos imágenes por país
- Formulario de contacto con validación básica
- Diseño responsive

## Palabras para probar en el buscador

- playa
- templo
- Japón
- Italia
- Brasil

## Publicación en GitHub Pages

1. Subir todos los archivos al repositorio de GitHub.
2. Entrar a Settings.
3. Ir a Pages.
4. En Branch seleccionar `main` y carpeta `/root`.
5. Guardar.
6. Copiar el enlace público generado por GitHub Pages.
